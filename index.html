Tools Programing

HTML

HTML (Hypertext Markup Language) is a markup language used to create the structure and content of web pages.

  • h1, h2, h3, p, ul, li
  • div, span, link, form
  • nav, section, footer
CSS

CSS (Cascading Style Sheets) is a language used to organize the appearance and style of web pages.

  • margin, padding, border, background
  • display, position, z-index, opacity
  • transform, transition, animation
Javascript

JavaScript is a programming language that is often used to create dynamic and responsive interactions on web pages.

  • let, const, array, object, if else, for
  • function, for in, for of, foreach
  • map, DOM, async, await
